Resource Profile: FinnishSchedulingPatient

Official URL: http://hl7.fi/fhir/StructureDefinition/FinnishSchedulingPatient Version: 0.2.0
Draft as of 2020-01-02 Computable Name: FinnishSchedulingPatient

Resource profile for patient to be used in Finnish social and healthcare setting. Supports one official and one temporary id. Supports home municipality extension.

Usage:

  • This Resource Profile is not used by any profiles in this Implementation Guide
