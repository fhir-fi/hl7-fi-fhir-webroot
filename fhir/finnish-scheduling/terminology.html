    Other Terminologies

    The Finnish core concepts for scheduling are defined in Sosiaali- ja terveydenhuollon ajanvarauspalvelujen ja palveluohjauksen käsitteet

    The content format for the appointment booking document (ajanvarausasiakirja) is defined in Ajanvarausasiakirjan tietosisältömäärittely.

    Relevant code sets in the national code server (besides the Ajanvarauksen tila described above) include

    • THL - Yhteystapa
    • THL - Viestintäkanava
    • Ajanvaraus - Yhteyshenkilön tyyppi

    There are many other terminologies used in Finland. Please see the list of some prominent ones in the terminology section of the Finnish FHIR Base Profiles specification.
