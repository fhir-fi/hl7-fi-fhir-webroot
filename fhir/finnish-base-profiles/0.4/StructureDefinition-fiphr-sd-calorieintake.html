Resource Profile: fiphr-sd-calorieintake

Official URL: http://phr.kanta.fi/StructureDefinition/fiphr-sd-calorieintake Version: 0.4.0
Draft as of 2021-02-22 Computable Name: FIPHRSDCalorieIntake

Finnish PHR profile for Calorie Intake observation resource

Finnish PHR profile for Observation used for measuring calorie intake. The profile is based on the Finnish PHR Nutrition profile.

Usage:

  • Examples for this Resource Profile: Observation/542ae72e-c3b0-404f-8c06-8f77441e89f7, Observation/542ae72e-c3b0-404f-8c06-8f77441e89f8
