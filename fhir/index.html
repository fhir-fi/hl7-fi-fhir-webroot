Implementation Guide Directory

This is the directory of FHIR implementation guides published and maintained by HL7 Finland.

Implementation guides published on this website

These implementation guides are actively being maintained.

  • Finnish Base Profiles

    This implementation guide specifies the Finnish FHIR base profiles.

    See the Finnish Base Profiles!

  • Finnish Implementation Guide for SMART App Launch

    This implementation guide specifies guidelines for and examples of using the SMART App Launch mechanism in Finland.

    See the Finnish Implementation Guide for SMART App Launch!

  • Finnish Scheduling

    The Finnish HL7 FHIR implementation guide for scheduling.

    See the Finnish Scheduling IG!

Older implementation guides in Simplifier

These are older implementation guides published by FL7 Finland in Simplifier.net FHIR publication platform and registry. It is foreseen that the scheduling related implementation guides are consolidated into a single implementation guide, with additional features, during 2023 and 2024.

  • FinnishScheduling R4

    Profiles for Finnish Scheduling (hl7.fi) resources, published in February 2020.

    See the FinnishScheduling R4.

  • FinnishAppointment

    Finnish Appointment Implementation Guide lists the profiled resources and describes the usage in online appointment scheduling scenarios. Published in September 2021

    See the FinnishAppointment.

  • Patient data to EHR HL7Fi IG

    This Implementation guide aims to outline the workflow and interactions needed in a generic use case where a self-care/wellness application needs to notify and send patient data to EHR system.

    See the Patient data to EHR HL7Fi IG.
